# st-link 下载链接

[安装器下载地址](https://www.st.com/en/development-tools/stsw-link004.html#get-software)

这个地址含有 st-link 驱动。

[github 地址](https://github.com/stlink-org/stlink.git)

这里是命令行工具集。
包括：

- st-flash： 一个可以利用 stlink 仿真器烧录 bin 文件到单片机 flash 的命令行工具。
- st-util： 一个 GDB 服务器。
